# Raining molecules

## Description

- Canvas drawing molecules raining in the screen

## Technologies

- HTML
- Stylus
- ES6 JavaScript

## Features

- Molecule-like cursor
- 5 types molecules
  - Benzene
  - Naphtalene
  - Diphenyle
  - Styrene
  - Coronene
